## Running application server 🎲

```sh
npx expo start
npx expo android
npx expo ios
npx expo web
```

## Installation ✅

```sh
npx create-expo-app -e with-router
npm i nativewind
npm i tailwindcss@3.3.2 --save-dev
npm i -D eslint @rocketseat/eslint-config
npm install -D prettier prettier-plugin-tailwindcss
```
